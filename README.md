# New-Chess-Master
New Chess Game
✅ 로그인 시스템

사용자 이름 입력 후 게임 시작
로그아웃 기능 포함

✅ Firebase 연동

실시간 데이터베이스에 점수 저장
자동으로 순위표 업데이트

✅ 게임 플레이

화살표 키(↑↓←→): 커서 이동
Enter 키: 말 선택 및 이동 확정
흰색 말만 조작 가능
검은색 말을 잡으면 +10점
검은색 King(♚)을 잡으면 +100점으로 승리!

✅ 실시간 순위표

상위 10명 표시
금/은/동 메달 표시
자동 업데이트

✅ 게임 통계

현재 점수
잡은 말 개수
이동 횟수

✅ 세련된 디자인

그라데이션 배경
애니메이션 효과
커서 펄스 효과
반응형 레이아웃
